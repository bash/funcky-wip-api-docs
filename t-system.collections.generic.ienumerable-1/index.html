    static Sequence.Successors<...TResult>(Option<TResult> first, (TResult -> Option<TResult>) successor) -> Self<TResult> 
     where ...TResult : notnull
    
            Generates a sequence based on a successor function stopping at the first None value.
            This is essentially the inverse operation of an Aggregate.
            
    Examples
    
            The sequence of all non-negative integers.
            
    Sequence.Successors(0, n => n + 1); // [0, 1, 2, ...]
    
            The fibonacci sequence.
            
    Sequence.Successors((0, 1), n => (n.Item2, n.Item1 + n.Item2)).Select(n => n.Item1); // [0, 1, 1, 2, 3, 5, 8, ...]
    
            The sequence of all inner exceptions.
            
    Exception exception = ...;
Sequence.Successors(exception, e => Option.FromNullable(e.InnerException)); // [exception, exception.InnerException, exception.InnerException.InnerException, ...]
    Remarks
    
        
    Use Skip on the result if you don't want the first item to be included.
