Option<TItem> Struct

readonly struct Option<TItem> : IComparable<Self>, IComparable, IOption, IEquatable<Self> 
     where TItem : notnull

Comparing Options: None values are always treated as being less than Some values. Some values are compared using the type's default comparer.
