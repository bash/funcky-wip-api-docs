Either<TLeft, TRight> Struct

readonly struct Either<TLeft, TRight> : IEquatable<Self>, IEither 
     where TLeft : notnull 
     where TRight : notnull

Remarks

Either values constructed using default are in an invalid state. Any attempt to perform actions on such a value will throw a NotSupportedException.
