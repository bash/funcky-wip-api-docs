    static readonly Self Discard.__
    A convenient shortcut for getting a Unit.
            Useful for using switch expressions purely with guards when matching on non-constant values
            or as an alternative to if / else if / else chains.
    Example
    using static Funcky.Discard;
return __ switch
{
    _ when user.IsFrenchAdmin() => "le sécret",
    _ when user.IsAdmin() => "secret",
    _ => "(redacted)",
};
    Remarks
    
        
    The name is intentionally two underscores as to not conflict with C#'s discard syntax.
