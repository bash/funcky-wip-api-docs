    static ActionToUnit<...T1, ..Tn>(((T1, ..Tn) -> void) action) -> ((T1, ..Tn) -> Unit)

    Converts an Action to a Func. This is useful when writing higher-level functions that want to generalize over both void and non-void functions.

    Example
    TResult RetryOnException<TResult>(Func<TResult> producer) { /* ... */ }

// We don't want to re-implement this function for
// actions, so we use `ActionToUnit` to bridge that gap:
void RetryOnException(Action action)
    => RetryOnException(ActionToUnit(action));

  • [Pure]

    static Fn<...T>(T value) -> T

    Infers a method group to its natural type i.e. Func<T, ...> and Action<T, ...>.

    Example

    Sometimes, the C# compiler is unable to infer a method group's natural type. 

    var pow = Curry(Math.Pow);
//        ^^^^^
// error CS0411: The type arguments for
// method 'Functional.Curry<...>' cannot
// be inferred from the usage.

    This function can sometimes help with that: 

    var pow = Curry(Fn(Math.Pow));
// or:
var pow = Fn(Math.Pow).Curry();
    Remarks

    This function is identical to Identity. Prefer the latter for all other cases.
